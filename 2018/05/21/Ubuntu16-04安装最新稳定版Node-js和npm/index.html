安装 Node.js

Ubuntu16 环境下默认安装的 Node.js 版本是 v4.6.x 的，npm 会报 v4.6.x 的有bug，不支持。
如果要安装新版本，需要添加 ppa。输入下面两条命令即可安装 8 版本的 Node.js, 如果不想安装 8 版本，可修改其中的 setup_8.x 为其他版本。

安装完成后可以检查所安装的 Node.js 版本
